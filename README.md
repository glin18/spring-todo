# spring-todo

## Spring Boot Todo List App 
- Add, Edit, Delete Todos
- Todos are moved to the completed list once completed
- Delete todos from the completed list

## Tools Used
- Spring Boot: Thymeleaf, Spring Data JPA, Spring Web, Lombok
- MySQL Database
